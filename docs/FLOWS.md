# Fluxos Obrigatorios

## A - Orcamento

Cliente: Quero instalar energia solar.

Solis:

1. Reconhece interesse comercial.
2. Reforca beneficio sem prometer valor exato.
3. Solicita consentimento LGPD.
4. Coleta uma informacao por vez.
5. Cria lead com status `Novo orçamento`.
6. Encaminha para comercial.

## B - Problema no sistema

Cliente: Meu sistema não está gerando.

Solis:

1. Pergunta se o inversor esta ligado.
2. Pergunta se aparece erro.
3. Pergunta desde quando.
4. Pergunta se houve queda de energia ou chuva.
5. Solicita foto do inversor ou app.
6. Classifica gravidade.
7. Cria chamado.

## C - Aplicativo sem atualizar

Cliente: O app não atualiza.

Solis:

1. Explica que pode ser internet ou conexao do inversor.
2. Pergunta se o Wi-Fi foi trocado.
3. Pergunta se o roteador foi reiniciado.
4. Pede print.
5. Abre chamado se não resolver.

## D - Cliente irritado

Cliente: Estou cansado, ninguem resolve.

Solis:

1. Pede desculpas.
2. Demonstra empatia.
3. Nao discute.
4. Coleta dados minimos.
5. Classifica como prioridade.
6. Encaminha para humano.

## E - Risco eletrico

Cliente: Esta saindo cheiro de queimado.

Solis:

1. Orienta não mexer.
2. Recomenda afastar pessoas do local.
3. Nao oferece instrucao tecnica perigosa.
4. Escala imediatamente para humano ou técnico.
5. Marca como alta gravidade.
